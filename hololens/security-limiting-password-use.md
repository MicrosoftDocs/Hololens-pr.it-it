# <span data-ttu-id="283a2-104">Limitazione dell'uso delle password</span><span class="sxs-lookup"><span data-stu-id="283a2-104">Limiting password use</span></span>

<span data-ttu-id="283a2-105">Nella maggior parte dei casi, i sistemi informatici di oggi usano le credenziali utente come base per la sicurezza e quindi dipendono di fatto dalle password riutilizzabili create dall'utente.</span><span class="sxs-lookup"><span data-stu-id="283a2-105">Most computer systems today utilize user credentials as the basis for security making them dependent on reusable, user-created passwords.</span></span> <span data-ttu-id="283a2-106">Di conseguenza, le password sono diventate anche la causa più comune della compromissione degli account e della violazione dei dati.</span><span class="sxs-lookup"><span data-stu-id="283a2-106">This has resulted in passwords also becoming the most common cause of account compromise and data breaches.</span></span> <span data-ttu-id="283a2-107">Ad esempio, le password possono essere intercettate durante la trasmissione o sottratte da un server (mediante attacchi di phishing o di tipo password spray) e compromesse per accedere a un account utente.</span><span class="sxs-lookup"><span data-stu-id="283a2-107">As an example of this, passwords can be intercepted in transmission or stolen from a server (by phishing or password spray attacks) and compromised to gain access to a user account.</span></span>
